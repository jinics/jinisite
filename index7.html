Coventry Map

Introduction

This project is to make a map for coventry uni. the target audiences are new first year students or people how are new to coventry. Mostly, the map shows the campus infomations and thing that will help to support students.

Concept

The inspiration was from a game that uses isometric as a graphic. From there, i have designed using isometric elements to look fun and active. In this way, the shapes are from real image so it is easy to find the building and infos.
