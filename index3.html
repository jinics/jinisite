Bookcover Design

Introduction

This project is to remake a bookcover for "A Short History of nearly Everything". This book is a easy science book for reader to get close to science.

Concept and Inspiration

I got inspiration from a bookshelf. When we go to bookstore, there are different section of each category. From that image, i try to make it more eyecatchging visual.

Design elements-colour

Choosing simple colour pattle, because the elements that have used in the design is bit compelcated, so i try to focus on the visual. Using two complementary color to make strong visual.
